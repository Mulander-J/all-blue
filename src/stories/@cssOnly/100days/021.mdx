## Learn

How to make a mouth

```html
	<svg>
		<circle cx="50" cy="50" r="25"/>
	</svg>
```
```css
.ring {
	fill: none;
	stroke: #F4D146;
}
.circle {
	fill: none;
	stroke: #F4D146;
	stroke-width: $radius;
	stroke-linecap: butt;
	stroke-dasharray: ($radius * 3.14);
}
.mouth {
  fill: none;
	stroke: #F4D146;
	stroke-width: $radius;
	stroke-linecap: butt;
  stroke-dashoffset: ($radius * 3.14 / 4);
  transform: rotate(45deg);
}
```
