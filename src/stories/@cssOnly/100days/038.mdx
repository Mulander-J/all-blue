## Code

```html
<div class="frame">
	<div class="logo" onclick="
        this.style.setProperty('--c1', 'hsla('+  Math.floor(Math.random()*360) +',100%,50%,1)');
        this.style.setProperty('--c2', 'hsla('+  Math.floor(Math.random()*360) +',100%,50%,1)');
        this.classList.toggle('active');
      ">
		<div class="prev"></div>
		<div class="next"></div>
	</div>
</div>
```
