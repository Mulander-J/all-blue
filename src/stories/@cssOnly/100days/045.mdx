## Code

```html
<div class="frame">
	<button class="btn_svg">
		<span>hover me</span>
		<svg class="base">
			<rect class="bg" />
			<rect class="line" />
		</svg>
	</button>
</div>
```

```css
.frame {
	position: absolute;
	top: 50%;
	left: 50%;
	width: 400px;
	height: 400px;
	margin-top: -200px;
	margin-left: -200px;
	border-radius: 2px;
	box-shadow: 1px 2px 10px 0px rgba(0, 0, 0, 0.3);
	overflow: hidden;
	background: #5ca4ea;
	color: #fff;
}
$lineH: 58;
$padY: 8;
$width: 200;
$gap: 50;
$c: ($lineH + $padY * 2 + $width) * 2;
button {
	position: absolute;
	top: 50%;
	left: 50%;
	transform: translate(-50%, -50%);
	cursor: pointer;
	appearance: none;
	border: none;
	position: relative;
	text-align: center;
	line-height: $lineH * 1px;
	font-size: 18px;
	letter-spacing: 1px;
	font-weight: 300;
	text-transform: uppercase;
	padding: $padY * 1px 12px;
	box-sizing: border-box;
	width: $width * 1px;
	color: #fff;
	background: none;
	.base {
		position: absolute;
		top: 0;
		left: 0;
		width: 100%;
		height: 100%;
		z-index: -1;
		.line {
			position: relative;
			width: 100%;
			height: 100%;
			fill: none;
			stroke-width: 2;
			stroke: #fff;
			stroke-dasharray: $gap $c;
			stroke-dashoffset: $gap;
			transition: all 0.8s ease;
		}
		.bg {
			width: 100%;
			height: 100%;
			fill: hsl(210, 77%, 64%);
			stroke: hsl(209, 100%, 78%);
			stroke-width: 2;
			transition: all 0.8s ease;
		}
	}
	&:hover {
		.line {
			stroke-dashoffset: $c * -1;
			z-index: -1;
		}
		.bg {
			fill: hsl(210, 65%, 58%);
		}
	}
}
```
