```html

.frame
	- for(let i = 0; i < 50; i ++)
		.dotRing
```

```css
$count: 50;

.frame {
	position: absolute;
	top: 50%;
	left: 50%;
	width: 400px;
	height: 400px;
	margin-top: -200px;
	margin-left: -200px;
	border-radius: 2px;
	box-shadow: 1px 2px 10px 0px rgba(0, 0, 0, 0.3);
	overflow: hidden;
	background: #27ae60;
}
.dotRing {
	position: absolute;
	width: 200px;
	height: 200px;
	top: 100px;
	left: 100px;
	border-radius: 50%;
	border: 2px dotted #fff;
	transform-origin: 50% 50%;
	animation: ring 10s ease both infinite;
	@for $i from 1 through $count {
		&:nth-child(#{$i}) {
			animation-delay: 0.2s * $i;
		}
	}
	&:nth-child(odd){
		border-style:dashed;
		border-color: #ff0;
	}
	&:nth-child(even){
		border-style:dotted;
		border-color: #fff;
	}
}
@keyframes ring {
	0% {
		transform: scale(0) translate(-200px, -200px);
		opacity: 0.4;
	}
	100% {
		transform: scale(3.5) translate(0px, 0px);
		opacity: 1;
	}
}
```
