## Learn

CSS `counter()`

```css
.parent {
  /*重置计数器*/
  counter-reset: key;
}
.elements::after {
  /* 增加计数器值 */
  counter-increment: key;
  /* 显示计数器 */
  content: "Label:" + counter(key);
}
```
