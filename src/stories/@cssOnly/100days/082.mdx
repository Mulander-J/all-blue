## Code

```html
<div class="frame">
	<input id="sender" type="checkbox" name="checker">
	<label class="flip" for="sender">
		<div class="front">send</div>
		<div class="back">done</div>
	</label>
</div>
```

```css
.frame {
	position: absolute;
	top: 50%;
	left: 50%;
	width: 400px;
	height: 400px;
	margin-top: -200px;
	margin-left: -200px;
	border-radius: 2px;
	box-shadow: 1px 2px 10px 0px rgba(0, 0, 0, 0.3);
	overflow: hidden;
	background: skyblue;
	color: #fff;
}
.flip {
	display: block;
	position: absolute;
	top: 37.5%;
	left: 37%;
	height: 12.5%;
	width: 26%;
	transform-style: preserve-3d;
	transition: all 0.8s ease;
	font-size: 20px;
	font-weight: 600;
	text-transform: uppercase;
	cursor: pointer;
}
input[type="checkbox"] {
	display: none;
}
#sender:checked ~ .flip {
	transform: rotateX(180deg);
}
.front,
.back {
	display: flex;
	justify-content: center;
	align-items: center;
	position: absolute;
	top: 0;
	left: 0;
	width: 100%;
	height: 100%;
	backface-visibility: hidden;
	border-radius: 25px;
	box-shadow: 4px 8px 12px 0 rgba(0, 0, 0, 0.1);
	text-shadow: -1px 3px 2px rgba(0, 0, 0, 0.3);
}
.front {
	background: #fff;
	color: #333;
}
.back {
	border: 2px solid #fff;
	background: #0f0;
	color: #fff;
	transform: rotateX(180deg);
}
```
