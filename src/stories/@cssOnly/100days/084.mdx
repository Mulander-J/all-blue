## Learn

[sass.list](https://sass-lang.com/documentation/modules/list/)

`append`

```
append($list, $val, $separator: auto) //=> list
```

Returns a copy of $list with `$val` added to the end.

If `$separator` is `comma`, `space`, or `slash`, the returned list is comma-separated, space-separated, or slash-separated, respectively. 

If it’s `auto` (the default), the returned list will use the same separator as `$list` (or space if `$list` doesn’t have a separator). 

Other values aren’t allowed.

	$list: ();
	@for $i from 1 through $n {
		$opt: if($i % 2 == 0, 0.2, 1);
		$list: append($list, rgba($c, $opt), comma);
	}
	@return $list;
}
